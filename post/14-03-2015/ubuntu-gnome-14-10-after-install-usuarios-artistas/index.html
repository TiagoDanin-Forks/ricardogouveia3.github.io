Ubuntu Gnome 14.10 after install para usuários artistas

Em geral, o designer trabalha no Mac OS. No Brasil, a gente dá um jeitinho.

Postado em 14-03-2015 – Tags: #ubuntu #after-install #linux

Introdução

Duas vezes por ano - em abril e em outubro - são lançadas novas versões do Ubuntu. Nessa época, os sites dedicados ao linux correm para lançar seu próprio “Ubuntu after install”: uma coleção de dicas, incluindo otimizações, programas e scripts, que melhoram a experiência de uso do sistema, adicionando e modificando funcionalidades.

Essa prática é que, para alcançar a maior quantidade possível de leitores, fornecem apenas dicas genéricas. Ou seja, quase nunca se vê programas de uso muito específico (por exemplo o Blender, para modelação 3D).

Falar de ferramentas para todos os esteriótipos de usuários linux resultaria em um artigo enorme e trabalhoso; resolvi então falar somente sobre como eu monto meu ambiente após uma formatação.

PS: Eu sou declaradamente um usuário “artista”. Descubra o seu tipo nesse artigo do site Linux Descomplicado.

Atualização

O primeiro passo após a instalar o Ubuntu sempre é a atualização dos programas e listas do sistema - em especial se você não selecionou “Baixar atualizações enquanto instala” no assistente de instalação.
Isso vai corrigir alguns erros de tradução, codecs multimídia, atualizações de programas e outras coisas.

sudo apt-get update && sudo apt-get upgrade

Tradução

Um dos bugs mais comuns é o GNOME vir metade em português e metade em inglês. Corrija executando:

sudo apt-get install language-pack-gnome-pt-base

Repositórios

Adicionar repositórios é uma tarefa arriscada, que se não feita com cuidado pode te fazer ter que formatar o computador (no pior caso).
No entanto, adicionar os repositórios oficiais dos programas que você usa em suas sources mantém seu sistema atualizado da maneira mais prática possível. A seguir, os repositórios que eu uso sempre que reinstalo meu S.O.:

Repositório (Comando add) Mantenedor
sudo add-apt-repository -y ppa:videolan/stable-daily VideoLAN/VLC
sudo add-apt-repository -y ppa:otto-kesselgulasch/gimp Não oficial/GIMP
sudo add-apt-repository -y ppa:gnome3-team/gnome3 Desenvolvedores GNOME
sudo add-apt-repository -y ppa:webupd8team/java WebUP8/Java
sudo add-apt-repository -y ppa:webupd8team/atom WebUP8/Atom editor
sudo add-apt-repository -y ppa:ricotz/docky Não oficial/Plank
sudo add-apt-repository -y ppa:satyajit-happy/themes Não oficial/Elegance Colors
sudo add-apt-repository -y ppa:libreoffice/ppa LibreOffice
sudo add-apt-repository -y ppa:numix/ppa Numix Project

Importante: depois de adicionar repositórios, execute sudo apt-get update para atualizar a lista de softwares e atualizações disponiveis.

Codecs e Descompactadores

Todo sistema precisa dos codecs para ser compatível com determinados tipos de arquivo de mídia. Com o Linux, ainda temos o agravante das licenças (não é possível incluir automaticamente determinados softwares proprietários por motivos legais).
Incluir eles após a instalação é fácil. Eles automaticamente se associarão a programas como VLC, gnome-video (totem), gnome-sushi (preview), mplayer e outros. Você pode adicioná-los com esse comando:

sudo apt-get install libxine1-ffmpeg mencoder flac faac faad sox ffmpeg2theora libmpeg2-4 uudeview libmpeg3-1 mpeg3-utils mpegdemux liba52-dev mpeg2dec vorbis-tools id3v2 mpg321 mpg123 libflac++6 totem-mozilla icedax lame libmad0 libjpeg-progs libdvdcss2 libdvdread4 libdvdnav4 libswscale-extra-2 ubuntu-restricted-extras

O descompactador de arquivos do Ubuntu é um excelente programa para usos básicos, porém você pode se deparar com arquivos compactados de formato incomum. O comando abaixo adiciona suporte a alguns formatos a mais:

sudo apt-get install unace unrar zip unzip p7zip-full p7zip-rar sharutils rar uudeview mpack arj cabextract and file-roller

Programas

Via repositório

Aqui a coisa começa a se especificar pros ‘artistas’. Esses são alguns programas úteis:

sudo apt-get install vlc gimp gimp-data gimp-plugin-registry gimp-data-extras inkscape blender bleachbit openjdk-7-jre oracle-java8-installer flashplugin-installer plank elegance-colors

Via script

Em geral, não se usa scripts para instalar programas no Ubuntu já que o terminal e os arquivos .deb resolvem a maior parte dos problemas.
Mas scripts podem ajudar, por exemplo, conseguindo a versão mais recente de um arquivo .deb da web com apenas um comando.

Para instalar o Google Chrome, por exemplo, copie e cole isso no seu terminal:

if [[ $(getconf LONG_BIT) = "64" ]]
then
echo "64bit Detected" &&
echo "Installing Google Chrome" &&
wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb &&
sudo dpkg -i google-chrome-stable_current_amd64.deb &&
rm -f google-chrome-stable_current_amd64.deb
else
echo "32bit Detected" &&
echo "Installing Google Chrome" &&
wget https://dl.google.com/linux/direct/google-chrome-stable_current_i386.deb &&
sudo dpkg -i google-chrome-stable_current_i386.deb &&
rm -f google-chrome-stable_current_i386.deb
fi

Esse script vai reconhecer se seu sistema é x32 ou x64 e instalar a versão adequada do navegador.

Para instalar o Adobe Air, use esse script do NoobsLab:

wget -O adobe-air.sh http://drive.noobslab.com/data/apps/AdobeAir/adobe-air.sh && chmod +x adobe-air.sh;sudo ./adobe-air.sh

Executáveis .deb

Algumas empresas, apesar de fazerem softwares para Linux, não disponibilizam repositórios. Apenas fornecem um link para download do executável em .deb.

Se você veio do windows, essa é a forma de instalação que você conhece. Clique 2x sobre o executável e ele abrirá, por padrão, na Central de Software do Ubuntu, e clique em instalar.

OBS1: se não for possível instalar, clique com o botão direito no arquivo e vá em Propriedades>Permissões, e verifique se a caixa Permitir executar como programa está marcada; se não estiver, marque-a.

OBS2: A Central de Software do Ubuntu é, na minha opinião, muito problemática e lenta. Se quer algo mais rápido e simples para instalar seus programas, use o gdebi (sudo apt-get install gdebi).

Abaixo, o download dos programas que eu uso em .deb:

  • Sublime Text: Editor de Texto para programadores
  • Dropbox: Arquivos em Cloud
  • Flareget: Gerenciador de Download
  • Brackets Editor: Outro editor, mais voltado para Desenvolvimento Web
  • Haroopad: Editor de Markdown
  • Google Web Design: Parecido com o Flash da Adobe, porém com saídas em HTML5

Ambiente GNOME

Se você conhece os ambientes de trabalho mais comuns do Linux, sabe que GNOME é um dois mais “bonitos”.
Porém, eu pessoalmente discordo que ele já venha num ‘design plug-and-play’. É preciso mudar algumas coisas antes.

Extensões do GNOME

Duas coisas que eu não gosto no GNOME são: o nome “Atividades” como botão principal do gnome (um ícone seria bem mais “clean”) e os hot-corners.
A extensão Activities Configurator serve para ajustar essas duas coisas. E isso é super fácil:

  1. Vá para a página de download da extensão usando o navegador Firefox
  2. Dê permissão para instalar a extensão
  3. Clique na chave ao lado do nome da extensão
  4. Confirme a instalação na caixa que aparecer.

Isso deve ser suficiente. Agora vamos configura-la:

  1. Abra a Ferramenta de ajustes e clique em extensões.
  2. Ative a extensão e clique no ícone ao lado do botão remover
  3. Ative a opção Hide text
  4. Clique em Select icon e selecione um arquivo .ico de sua preferência para ser o ícone do menu. Você pode baixar o que eu uso aqui
  5. Defina Icon padding para o menor valor possível
  6. Ative a opção Disable hot corner
    Pronto. Se seu ícone ficar muito longe do nome do aplicativo a direita, aperte alt + F2, digite r e pressione enter. Isso vai recarregar as configurações do gnome-shell.

OBS3: Se, da próxima vez que você iniciar o computador, estas configurações não estiverem ativadas, mude seu ambiente de gnome para default na tela de logon.

Elegance Themes

Se você instalou todos os programas que eu sugeri na seção Via repositório, você já tem o Elegance Colors. Esse programa na verdade é um tema altamente configurável, com várias configurações pré-definidas.

Para ativa-lo, digite no terminal: 

elegance-colors && gsettings set org.gnome.shell.extensions.user-theme name 'elegance-colors'

Se o visual do seu GNOME não mudar imediatamente:

  1. Vá na Ferramenta de ajustes e selecione Extensões
  2. Ative a extensão User themes
  3. Clique em Aparência
  4. Na opção Tema do shell selecione Elegance-colors

Agora, configure o Elegance:

  1. Caso não esteja já aberto, procure por Elegance no menu e o abra.
  2. Em General Settings, na opção Load config from preset, selecione uma das opções e clique em apply

Dica: as melhores opções são: Ubuntu touch e Numix.

Numix GTK theme/Icons

Os ícones nativos do gnome são uma bosta muito detalhados e diferentes.
Os pacotes de ícones Numix são padronizados e cobrem uma enorme quantidade de aplicações. Para instalar, o comando é:

sudo apt-get install numix-icon-theme-circle numix-gtk-theme

Para ativar:

  1. Vá em ferramenta de ajustes > Aparência
  2. Selecione Numix em Janelas e GTK+
  3. Selecione Numix-circle em Ícones

Ferramenta de Ajustes (finais)

A ferramenta de ajustes (gnome-tweak-tool) é onde você gerencia extensões, temas e outros ajustes da interface. Você já acessou ela várias vezes.

Nas versões mais recentes do Ubuntu GNOME ela já vem instalada por padrão.

Como ajuste final, a barra superior fica desproporcional ao resto da interface com o tema Ubuntu-touch e Numix do Elegance. Para corrigir isso:

  1. Vá em Ferramentas de Ajustes>Fontes
  2. Mude Fator de escala para 0,90

Parabéns, seu ambiente está de muito bom gosto agora.

Limpando

E como o tio do Jack Chan diria: Mais uma coisa.

As vezes, fazer todas essas modificações deixa alguns pacotes inúteis no seu sistema.
O Linux reconhece esses pacotes, mas ele aguarda você sinalizar que deseja removê-los.
Para limpar seu sistema, execute: 

echo "Cleaning Up" &&
sudo apt-get -f install &&
sudo apt-get autoremove &&
sudo apt-get -y autoclean &&
sudo apt-get -y clean

That’s all folks.
